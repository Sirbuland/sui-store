
## :star: Get Started

```bash
# install expo cli
npm install -g expo-cli
```
``` bash
# install dependencies
npm install
```
Or use `yarn`
``` bash
yarn install
```
``` bash

# run project
expo start
```
``` bash
Change the API in config.js to the server that you're running. 
If you use my backend project the IP should be the IPv4 Address on your computer (cmd -> ipconfig)
```
